# 🚀 Crypto Alert Telegram Bot

A real-time crypto alert system that notifies you on Telegram when prices hit your conditions.

## 🔥 Features

- 📊 Live crypto prices (Binance API)
- 🔔 Price alerts (above / below)
- 🔁 Repeat alerts with cooldown control
- ⚡ Optimized (1 API call per coin)
- 👥 Multi-user support
- 💾 Persistent alerts (saved automatically)

---

## 🤖 Try the Bot

👉 http://t.me/sidi_crypto_bot

---

## 🧠 How It Works

Users set alerts like:
/set BTCUSDT above 70000
/set ETHUSDT below 2000


The bot:
1. Tracks prices in real-time
2. Checks conditions
3. Sends alerts instantly on Telegram

---

## 📌 Commands


/start - Start the bot
/help - Show help menu

/set BTCUSDT above 70000
/set ETHUSDT below 2000

/list - View your alerts
/delete 1 - Delete alert

/repeat 1 on - Enable repeat
/repeat 1 off - Disable repeat

/cooldown 1 60 - Set cooldown (seconds)

/price BTCUSDT - Get current price
